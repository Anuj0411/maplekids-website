Language Switcher Layout Test

This page tests the side-by-side layout of the language switcher buttons.

Desktop Layout Test

Expected: Two buttons side by side horizontally

Mobile Layout Test

Expected: Two buttons side by side, responsive

Instructions

  1. Check that both buttons are displayed side by side (horizontally)
  2. Verify that the flag icons and text codes are visible
  3. Test hover effects by hovering over the buttons
  4. Ensure the layout works on different screen sizes

CSS Properties Applied

  • display: flex !important - Forces flex layout
  • flex-direction: row !important - Ensures horizontal layout
  • justify-content: center - Centers the buttons
  • gap: 8px - Adds space between buttons
  • display: inline-flex !important - Makes buttons inline
  • white-space: nowrap - Prevents text wrapping
